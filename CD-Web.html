BOREAS CD-ROM Web Simulation


The BOREAS Information System (BORIS) is in the process of creating a set of CD-ROMs to hold all of the finalized BOREAS data. These CD-ROMs will all use a HyperText Markup Language (HTML) interface -- looking and operating just like a Web site, but based on a CD-ROM (not requiring any Internet connection). In order to get some design feedback from BOREAS Investigators, we have placed the currently available files onto a Web server for people to use and evaluate.
We invite you to browse through the following pages, see if you like the design, the indexes, the way the data files are linked -- anything. We are looking for feedback to improve the design, so if you have any comments or questions, please contact David Landis (David.R.Landis@gsfc.nasa.gov).

Things to Note:

  • This CD-ROM Web simulation uses some JavaScript. You must use version 4 (or higher) of either Netscape or Internet Explorer to make this site function correctly.
  • The server for the CD-ROM Web simulation site is a bit slow. The final CD-ROMs will look and operate just like this site, only much faster.
  • This site does not have a complete collection of BOREAS data sets -- these are just the ones that were processed to completion first. More data sets will be added as they are compeleted.
  • This site is password protected, using the current BOREAS Data Password. If you want to get BOREAS data, and are not a BOREAS Investigator, you can go to the ORNL DAAC site to get BOREAS data.
