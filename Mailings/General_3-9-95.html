    E-Mail Aliases For the Science Teams
    
Based on the feedback we received about the initial E-mail capabilities, we
have added a set of E-mail aliases (or 'exploders' as some people call them)
for use.  We have tested the staff lists successfully and except for some
possible missed syntax errors, the other lists should work as well.

The E-mail aliases that have been established include:

AFM - Addresses for all known BOREAS AFM participants
HYD - Addresses for all known BOREAS HYD participants
RSS - Addresses for all known BOREAS RSS participants
TE  - Addresses for all known BOREAS TE participants
TF  - Addresses for all known BOREAS TF participants
TGB - Addresses for all known BOREAS TGB participants

BOREAS_STAFF - Addresses for all GSFC BOREAS staff personnel
BORIS_STAFF  - Addresses for all GSFC BORIS staff personnel


To use these, send your message to ALIAS@boreas.gsfc.nasa.gov where ALIAS is
any of the ones listed above.  Any delivery difficulties, bad addresses,
etc. will be returned to a system account on the boreas.gsfc.nasa.gov
system.  We will check the account regularly to be sure that any mail
troubles are settled as quickly as possible.


    

    WWW Access
    
BORIS has established a preliminary set of BOREAS World-Wide Web pages.
They can be viewed with any Web browser program (such as NCSA Mosaic or
Netscape) connected to the Internet. The pages are located at:
http://boreas.gsfc.nasa.gov/

They contain a description of the BOREAS project, the Science Teams,
Preliminary Results for 1994, and digitized photographs from the various
sites.  They will be expanding and changing as we add more photographs,
more links, more maps, more figures, and other things.  The current
long-term plan is to also provide access to the 'cleaned up' BOREAS data
via this means.

If you would like to send comments on the BOREAS pages, or find any glaring
errors, please e-mail David Landis at "landis@ltpmail.gsfc.nasa.gov". If
you know of any related WWW pages that we should link to and/or if you have
any digital maps of the BOREAS region (any scale, including site maps) and
would like to contribute them, please send e-mail to David with that
information as well (please include the complete URL address of the
starting page and where it should go in the BOREAS pages).
